## ACP 协议通信流程

```
┌──────────────────────────────────────────────────────────┐
│ ACP (Agent Communication Protocol)                       │
│ 基于 JSON-RPC 2.0 的 WebSocket 双向通信                 │
└────────────────┬─────────────────────────────────────────┘
                 │
                 ▼
        ┌────────────────────────┐
        │ AgentMessagingService  │
        │ _sendViaACPProtocol()  │
        └────────────┬───────────┘
                     │
                     ▼
        ┌───────────────────────────────────┐
        │ _getOrCreateACPConnection()       │
        │ 获取或创建WebSocket连接           │
        └────────────┬──────────────────────┘
                     │
                     ▼
        ┌──────────────────────────────┐
        │ ACPAgentConnection           │
        │ connect(wsUrl, token)        │
        └────────────┬─────────────────┘
                     │
                     ▼
        ┌──────────────────────────────┐
        │ WebSocket建立连接            │
        │ 启动心跳 (30秒间隔)          │
        │ 启动接收线程                │
        └────────────┬─────────────────┘
                     │
                     ▼
        ┌──────────────────────────────┐
        │ 创建 ActiveTask              │
        │ registerTaskCallbacks()      │
        │ _taskCallbacks[taskId]       │
        └────────────┬─────────────────┘
                     │
                     ▼
        ┌──────────────────────────────┐
        │ connection.agent.chat()      │
        │ 发送JSON-RPC 2.0请求:       │
        │ {                            │
        │   "jsonrpc": "2.0",         │
        │   "method": "agent.chat",   │
        │   "params": {...},          │
        │   "id": requestId           │
        │ }                            │
        └────────────┬─────────────────┘
                     │
                     ▼
        ┌──────────────────────────────┐
        │ 创建taskCompleter            │
        │ 等待任务完成                │
        │ taskCompleter.future         │
        │ .timeout(300秒)             │
        └────────────┬─────────────────┘
                     │
        ┌────────────┼────────────────────────────┐
        │ 并行处理：                  │           │
        │                           │           │
        ▼                           ▼           ▼
    ┌─────────────────┐    ┌──────────────┐   ┌──────────┐
    │接收来自Agent的  │    │ 等待事件      │   │ 监控超时 │
    │通知/消息:       │    │ task.started │   │ 及取消   │
    │                 │    │ ui.*         │   │          │
    │- task.started   │    │ task.* → 解│   │ 300秒超时│
    │- ui.textContent │    │   析回调     │   │ →异常    │
    │- ui.*           │    │ task.error → │   │ →异常    │
    │- task.completed │    │   taskComplete│  │          │
    │- task.error     │    │   er.complete│  │ cancel() │
    │                 │    │ task.completed│  │ →中断    │
    │ → _handleInco*  │    │   → taskComple│  │          │
    │ → 分配回调处理  │    │   ter.complete│  │          │
    │                 │    └──────────────┘   └──────────┘
    └────────────┬────┘
                 │
        ┌────────┴────────────┐
        │ 回调事件处理         │
        │                     │
        ├─────────────────┐   │
        │ onTextContent   │   │
        │ → activeTask    │   │
        │ .onStreamChunk()│   │
        └─────────────────┘   │
        │                     │
        ├─────────────────┐   │
        │ onActionConfirm │   │
        │ → activeTask    │   │
        │ .onActionConfirm│   │
        │ ation()         │   │
        └─────────────────┘   │
        │                     │
        ├─────────────────┐   │
        │ onSingleSelect  │   │
        │ → activeTask    │   │
        │ .onSingleSelect │   │
        │ ()              │   │
        └─────────────────┘   │
        │                     │
        └─────────────────────┘
                 │
                 ▼
        ┌──────────────────────────┐
        │ taskCompleter.complete() │
        │ 或                       │
        │ timeout() / error()      │
        │ 触发await返回           │
        └────────────┬─────────────┘
                     │
                     ▼
        ┌──────────────────────────┐
        │ 构建响应Message          │
        │ - content: 累积文本      │
        │ - metadata: 元数据       │
        │ - trace_id: taskId       │
        └────────────┬─────────────┘
                     │
                     ▼
        ┌──────────────────────────┐
        │ 清理回调                 │
        │ unregisterTaskCallbacks()│
        └────────────┬─────────────┘
                     │
                     ▼
        ┌──────────────────────────┐
        │ 返回响应Message          │
        └──────────────────────────┘
```
