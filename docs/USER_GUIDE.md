## 快速开始

### 首次启动应用

1. **设置主密码**
   - 应用首次启动时，需要设置一个主密码
   - 主密码用于保护你的所有敏感数据（API 密钥、聊天记录等）
   - 建议使用 8 个以上字符，包含大小写字母和数字

2. **启用生物识别（可选）**
   - 完成主密码设置后，你可以启用 Face ID、Touch ID 或指纹认证
   - 这样每次打开应用时无需输入密码

3. **配置你的第一个 Agent**
   - 进入主页，点击 **"添加 Agent"** 按钮
   - 选择 **本地 LLM 代理** 或 **远端 Agent**（见下文）

## 基础功能

### 一、添加和管理 Agent

#### 1.1 本地 LLM 代理
本地代理直接在你的设备上运行 AI 模型，无需网络连接。

**支持的 AI 提供商：**
- **OpenAI** - GPT-4、GPT-4o、GPT-3.5 Turbo（需要 API 密钥）
- **Anthropic Claude** - Claude 3、Claude 2（需要 API 密钥）
- **Google Gemini** - Gemini Pro、Gemini Vision（需要 API 密钥）
- **DeepSeek** - DeepSeek API（需要 API 密钥）
- **Ollama** - 本地部署的模型（无需网络）
- 其他支持 OpenAI 兼容 API 的提供商

**配置步骤：**
1. 点击 **"+ 添加 Agent"** → **"本地 LLM 代理"**
2. 输入 Agent 名称（如 "ChatGPT"）
3. 选择 LLM 提供商
4. 输入 API 密钥（或 Ollama 服务器地址）
5. 选择模型和参数（温度、最大 Token 数等）
6. 点击 **"保存"**

#### 1.2 远端 Agent（通过 ACP 协议）
远端代理运行在远程服务器上，通过 ACP 协议与你的设备通信。

**配置步骤：**
1. 点击 **"+ 添加 Agent"** → **"远端 Agent"**
2. 输入连接信息：
   - **Agent ID**：唯一标识符
   - **Server URL**：Agent 服务器地址（如 `ws://192.168.1.100:8080`）
   - **Token**：服务器提供的认证令牌
3. 点击 **"测试连接"** 验证配置
4. 点击 **"保存"**

**远端 Agent 的优势：**
- 支持系统工具（文件操作、进程执行等）
- 支持自定义技能包
- 适合企业部署和团队协作

### 二、聊天与消息

#### 2.1 开始对话
1. 在主页选择一个 Agent
2. 在输入框输入你的问题或指令
3. 点击 **发送** 或按 `Enter` 键
4. Agent 将实时流式响应你的请求

#### 2.2 富文本消息功能
在聊天时，你可以添加：
- **文本** - 支持 Markdown 格式
- **图片** - 点击 📷 按钮上传（支持 JPG、PNG）
- **文件** - 点击 📎 按钮上传任何文件
- **语音** - 点击 🎤 按钮录制语音消息
- **表情符号** - 点击 😊 按钮选择表情
- **@提及** - 在群组聊天中使用 `@agent名称` 来指定特定 Agent

#### 2.3 消息操作
在任何消息上，你可以：
- **复制** - 复制消息内容到剪贴板
- **回复** - 引用该消息进行回复
- **删除** - 删除该消息
- **搜索** - 使用搜索功能查找特定消息

#### 2.4 会话管理
- **新建会话** - 点击 **"+ 新会话"** 开始新的对话
- **查看历史** - 左侧栏显示所有历史会话
- **删除会话** - 长按会话名称，选择删除
- **搜索消息** - 使用搜索栏快速查找历史对话

### 三、群组聊天

群组聊天允许你与多个 Agent 同时协作。

#### 3.1 创建群组
1. 点击 **"+ 创建群组"**
2. 输入群组名称
3. 选择 **Admin Agent**（协调者）和其他 **成员 Agent**（至少 2 个）
4. 选择编排模式（见下文）
5. 点击 **"创建"**

#### 3.2 三种群组编排模式

**模式 1：标准模式（轮流对话）**
- Admin Agent 协调多个成员 Agent 依次参与讨论
- 适合需要多角度分析的任务
- 可配置最大轮次（默认 50 次）

**模式 2：Planning Mode（计划审核）**
- Admin 生成详细的 JSON 执行计划
- 你可以在 UI 上审核、修改或跳过任务
- 批准后自动执行计划
- 适合需要用户干预的自动化流程

**模式 3：Flow Mode（自动化工作流）**
- Admin 生成分阶段的执行计划（Stage 1 → Stage 2 → ...）
- 系统自动驱动各阶段按顺序执行
- 你可以随时暂停、恢复、跳步或中止流程
- 适合复杂的多步骤工作流

#### 3.3 在群组中发送消息
1. 选择群组进入聊天界面
2. 在输入框输入消息
3. 使用 `@agent名称` 指定要回复的特定 Agent
4. 点击发送

#### 3.4 管理执行计划（Planning/Flow Mode）
- **审核任务** - 查看 Admin 生成的计划，每个任务可单独审核
- **修改任务** - 点击任务卡片编辑内容
- **跳过任务** - 标记某个任务为跳过
- **确认执行** - 所有修改完成后点击 **"确认并执行"**
- **暂停/恢复** - 在执行过程中可随时暂停或继续
- **查看结果** - 每个任务完成后显示实时结果

## 高级功能

### 一、多模态路由

为不同类型的内容自动选择最优的 AI 模型。

**配置步骤：**
1. 进入 **设置** → **多模态配置**
2. 为以下内容类型分别选择 Agent：
   - **文本处理** - 如 GPT-4
   - **图像分析** - 如 GPT-4 Vision 或 Claude Vision
   - **音频处理** - 支持的语音模型
   - **视频处理** - 支持的多模态模型
3. 当你发送包含这些内容类型的消息时，系统自动路由到对应的 Agent

### 二、系统工具和技能包

#### 2.1 远端 Agent 的系统工具
远端 Agent 支持调用本地系统工具：
- **文件操作** - 读取、写入、删除文件
- **进程执行** - 运行系统命令（如 bash、Python）
- **系统信息查询** - 获取 CPU、内存、磁盘等信息

**如何使用：**
- 直接在聊天中要求 Agent 执行系统操作，如"在桌面创建一个文件"
- 系统会弹出权限确认对话框（如果需要）
- 批准后 Agent 执行操作

#### 2.2 导入和管理技能包
技能包是自定义功能集，可以扩展 Agent 的能力。

**导入技能包：**
1. 进入 Agent 详情 → **"技能包"** 标签
2. 点击 **"导入技能包"**
3. 选择来源：
   - **本地文件** - 上传 ZIP 文件
   - **URL** - 输入远程技能包 URL
4. 点击 **"导入"**
5. 导入后，Agent 自动获得新的技能

### 三、权限管理与审计

Shepaw 对所有敏感操作进行权限控制。

#### 3.1 权限等级
- **SAFE** - 安全操作，自动执行
- **WARNING** - 需要确认的操作
- **DANGEROUS** - 高风险操作，需要明确批准

#### 3.2 查看权限历史
1. 进入 **设置** → **权限与审计** → **权限历史**
2. 查看所有 Agent 请求的操作记录
3. 每条记录显示：
   - 操作描述
   - Agent 名称
   - 时间戳
   - 批准状态（已批准/已拒绝）

#### 3.3 推理日志
1. 进入 **设置** → **推理日志审计**
2. 查看所有 AI 调用的统计数据：
   - Token 消耗统计
   - 平均响应时间
   - 错误记录
   - 按 Agent 分类的详细日志
3. 可导出为 CSV 或 JSON 格式

## 安全与隐私

### 一、密码和生物识别

#### 1.1 设置主密码
1. 进入 **设置** → **安全** → **密码**
2. 点击 **"设置主密码"**
3. 输入新密码（建议 8 个以上字符）
4. 确认密码

#### 1.2 修改密码
1. 进入 **设置** → **安全** → **密码**
2. 点击 **"修改密码"**
3. 输入当前密码
4. 输入新密码
5. 确认新密码

#### 1.3 启用生物识别
1. 进入 **设置** → **安全** → **生物识别**
2. 点击 **"启用 Face ID"** 或 **"启用指纹"**
3. 按照系统提示完成设置
4. 下次打开应用时可使用生物识别解锁

### 二、本地数据存储

所有数据默认存储在你的设备上：
- **聊天记录** - 本地 SQLite 数据库
- **Agent 配置** - 加密存储的 API 密钥
- **用户设置** - 本地配置文件

**隐私特点：**
- 无云同步（除非你主动导出）
- 离线可用（仅本地 Agent）
- 完整的数据控制权

### 三、数据导出与备份

#### 3.1 导出数据
1. 进入 **设置** → **数据管理** → **导出数据**
2. 选择导出格式：
   - **JSON** - 完整数据备份
   - **CSV** - 仅聊天记录
3. 点击 **"导出"**
4. 选择保存位置

#### 3.2 导入备份
1. 进入 **设置** → **数据管理** → **导入数据**
2. 选择备份文件（JSON 格式）
3. 点击 **"导入"**
4. 应用将还原所有数据

### 四、删除数据

#### 4.1 删除单个会话
- 在会话列表中长按会话名称
- 选择 **"删除"**

#### 4.2 删除所有数据
1. 进入 **设置** → **数据管理** → **清除所有数据**
2. 输入确认密码
3. 点击 **"确认删除"**
4. 警告：此操作不可逆

## 常见问题

### Q1: 如何连接本地 Ollama 模型？
**A:** 
1. 在本地计算机安装 Ollama（ollama.ai）
2. 在 Ollama 中下载模型：`ollama pull llama2`
3. 在 Shepaw 中添加本地 LLM Agent，选择 Ollama
4. 输入服务器地址：`http://localhost:11434`
5. 点击 **"测试连接"** 验证

### Q2: Planning Mode 和 Flow Mode 的区别？
**A:** 
- **Planning Mode**：AI 生成计划，你在 UI 上逐个审核和修改任务，然后执行
- **Flow Mode**：AI 生成分阶段计划，系统自动执行所有阶段，你可随时暂停或中止

Planning Mode 适合需要人工干预的复杂任务，Flow Mode 适合完全自动化的工作流。

### Q3: 如何提高聊天速度？
**A:**
- 使用本地 LLM 模型（如 Ollama）避免网络延迟
- 减少消息中的文件大小（缩小图片、压缩文件）
- 使用流式响应（默认启用）
- 关闭不必要的多模态分析

### Q4: 如何确保 API 密钥安全？
**A:**
- 所有 API 密钥使用主密码加密存储在本地
- 永远不要分享你的主密码或备份文件
- 定期更换 API 密钥
- 使用强密码（8+ 字符，包含大小写和数字）

### Q5: 支持离线使用吗？
**A:**
- 本地 LLM Agent（如 Ollama）完全离线
- 云端 Agent（OpenAI、Claude 等）需要网络连接
- 查看历史聊天记录和设置无需网络

### Q6: 如何删除 Agent？
**A:**
1. 进入 Agent 列表
2. 长按要删除的 Agent
3. 选择 **"删除"**
4. 确认删除（聊天记录会被保留）

### Q7: 群组聊天中 Admin 的作用是什么？
**A:**
Admin Agent 负责：
- 分析用户需求
- 生成执行计划（Planning/Flow Mode）
- 协调其他 Agent 的工作
- 整合最终结果

### Q8: 如何更新 Agent 配置？
**A:**
1. 进入 Agent 详情页面
2. 点击 **"编辑"** 按钮
3. 修改配置（API 密钥、模型、参数等）
4. 点击 **"保存"**

### Q9: 支持哪些文件格式？
**A:**
- **图片**：JPG、PNG、GIF、WebP
- **文档**：PDF、TXT、Word（.docx）、Excel（.xlsx）
- **代码**：所有文本格式（.py、.js、.java 等）
- **其他**：ZIP、视频（部分）、音频

### Q10: 如何处理 Agent 连接失败？
**A:**
1. 检查网络连接（远端 Agent）或服务状态（Ollama）
2. 验证 Server URL 和 Token 是否正确
3. 点击 **"测试连接"** 诊断问题
4. 查看应用日志获取更多错误信息
